# Lab 01: Read A Proto Contract

## Task

Open [../EXAMPLES/greeter/greeter.proto](../EXAMPLES/greeter/greeter.proto) and identify:

- package
- service
- unary method
- streaming method
- request messages
- response messages
- field numbers
- generated-code option

## Questions

1. Which field numbers are already used?
2. Which method is unary?
3. Which method is server streaming?
4. What would break if field `1` changed meaning?
5. What generated client/server boundary would be created?

## Done When

You can explain the proto without saying “it is just a DTO.”

# gRPC System Design Prompts

Practice these out loud.

1. Design a payment gRPC API for checkout.
2. Design an inventory reservation gRPC service.
3. Design a streaming order-status watch API.
4. Design a schema governance process for 200 services.
5. Deploy gRPC behind Envoy in Kubernetes.
6. Add gRPC-Web support for browser clients.
7. Build observability and SLOs for gRPC services.
8. Design a retry/deadline policy for internal RPCs.
9. Debug a proto breaking-change incident.
10. Debug a load-balancing incident during rollout.

For each answer, include contract, runtime, security, deployment, observability, and failure modes.

# Project 01: Product Catalog GraphQL API

## Outcome

Design and document a product catalog GraphQL API.

## Deliverables

- SDL schema
- named query operations
- mutation payloads
- cursor pagination
- resolver map
- error code list
- README with sample operations

## Acceptance Criteria

- product list is paginated
- all operations are named
- mutation inputs and payloads are typed
- schema does not expose database table names directly
- nullable vs non-null fields are justified

## Interview Proof

```text
I can design a GraphQL product API with clean schema contracts, operations, resolver boundaries, pagination, and error behavior.
```

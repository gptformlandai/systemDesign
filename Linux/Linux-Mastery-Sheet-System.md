# Linux Mastery Sheet System - Start Here

This folder is a modular beginner-to-pro Linux mastery track.

Start with:

- [Linux-Mastery-Track-Index.md](Linux-Mastery-Track-Index.md)
- [linux-mastery-lab/README.md](linux-mastery-lab/README.md)
- [linux-mastery-lab/LEARNING_PATH.md](linux-mastery-lab/LEARNING_PATH.md)

The goal:

```text
Linux mastery = understand operating-system behavior, use commands as evidence, and troubleshoot production systems safely
```

The modular track has:

- `01-Foundations` for Linux mental model, filesystem, users, permissions, shell, and environment.
- `02-Command-Line-Practical` for files, text processing, processes, packages, services, networking, and storage.
- `03-Senior-Production` for boot, systemd, performance, containers, security, observability, automation, and advanced diagnostics.
- `04-Scenario-Practice` for web server, resource, network, permission, container, on-call, and cloud VM scenarios.
- `05-Special-Interview-Rounds` for interview Q&A, command maps, anti-patterns, and debugging traps.
- `06-Practice-Upgrade` for active recall, command drills, mini projects, and production readiness.
- `linux-mastery-lab` for safe diagnostic scripts, labs, projects, cheatsheets, interview prep, and runbooks.

Core mental model:

```text
symptom + command evidence + OS layer + safe mitigation + prevention = Linux production skill
```

Use the root index as the source of truth for study order.

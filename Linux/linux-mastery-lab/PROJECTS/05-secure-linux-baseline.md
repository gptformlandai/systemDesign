# Project 05: Secure Linux Baseline

Document a secure baseline for a Linux server.

Include:

- user and sudo review
- SSH hardening
- open port review
- service user review
- patching plan
- firewall/security group rules
- audit logs
- secrets/config file permissions
- backup and restore validation

Deliverable:

```text
risk -> current state -> recommended fix -> validation command -> rollback plan
```

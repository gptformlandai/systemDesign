# Linux Interview Questions

## Foundation

1. What is Linux?
2. Kernel vs shell?
3. What is a process?
4. What are `/proc` and `/sys`?
5. What does PATH do?

## Practical

1. How do you find files and search logs?
2. How do you debug permission denied?
3. How do you check service status and logs?
4. How do you find listening ports?
5. How do you debug disk full?

## Senior

1. Explain Linux boot flow.
2. Explain load average.
3. Explain OOM killer.
4. Explain cgroups and namespaces.
5. Explain SELinux/AppArmor.

## Scenarios

1. Web server is down but SSH works.
2. Host has high load.
3. Service cannot call dependency.
4. Container is OOMKilled.
5. Patching caused a reboot failure.

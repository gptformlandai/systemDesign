# Linux Command Map Cheatsheet

| Goal | Commands |
|---|---|
| host identity | `uname -a`, `hostnamectl`, `uptime` |
| user identity | `id`, `whoami`, `groups` |
| files | `ls`, `find`, `stat`, `du`, `df` |
| text/logs | `grep`, `awk`, `sed`, `sort`, `uniq`, `tail` |
| processes | `ps`, `top`, `pgrep`, `kill`, `lsof` |
| services | `systemctl`, `journalctl` |
| network | `ip`, `ss`, `curl`, `getent`, `dig`, `nc`, `ssh -v` |
| storage | `lsblk`, `findmnt`, `df`, `du`, `lsof +L1` |
| security | `sudo -l`, `namei -l`, `getenforce`, `aa-status` |
| advanced diagnostics | `strace`, `lsof`, `tcpdump`, `perf`, `ulimit`, `sysctl`, `coredumpctl` |

# Project 02: Semantic Product Search

Goal: design semantic search for an ecommerce catalog.

Build:

- product embeddings
- category, brand, price, inventory, and region metadata
- exact keyword fallback plan
- business reranking plan

Queries:

- comfortable shoes for standing all day
- waterproof jacket for hiking
- gift for coffee lover

Interview talking points:

- dense vs sparse retrieval
- inventory filters
- personalization
- conversion and relevance metrics

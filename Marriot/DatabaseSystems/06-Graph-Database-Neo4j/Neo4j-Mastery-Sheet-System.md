# Neo4j Mastery Sheet System - Start Here

This folder is a modular beginner-to-pro Neo4j graph database track.

Start with:

- [Neo4j-Interview-Track-Index.md](Neo4j-Interview-Track-Index.md)
- [neo4j-mastery-lab/README.md](neo4j-mastery-lab/README.md)
- [neo4j-mastery-lab/LEARNING_PATH.md](neo4j-mastery-lab/LEARNING_PATH.md)

The modular track has:

- `01-Starter-Path` for fundamentals, graph mental models, setup, Cypher basics, CRUD, and simple traversals.
- `02-Intermediate-Backend` for Cypher, graph modeling, constraints/indexes, transactions, drivers, import/sync, and query plans.
- `03-Senior-MAANG` for graph algorithms, clustering, security, backup, scale patterns, GraphRAG, cloud, Kubernetes, testing, and observability.
- `04-Scenario-Practice` for recommendations, fraud, identity graphs, knowledge graphs, permission graphs, supply chain, lineage, dependency graphs, system design cases, and database tradeoff analysis.
- `05-Special-Interview-Rounds` for anti-patterns, internals, debugging, and interview Q&A.
- `06-Practice-Upgrade` for active recall, runnable labs, mini projects, cheat sheets, roadmap, golden rules, access-control drills, blast-radius drills, and pro design review.
- `neo4j-mastery-lab` for local Docker/cypher-shell practice, sample graph data, scripts, projects, runbooks, and interview prep.

Core mental model:

```text
Neo4j mastery = domain question + graph model + Cypher traversal + constraints/indexes + query plan + bounded fan-out + operations
```

Use the root index as the source of truth for study order.

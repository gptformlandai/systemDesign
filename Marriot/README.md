## Suggested Study Order

```text
1. Java
2. Python
3. SQL
4. SpringBoot
5. Microservices
6. DSA daily practice in parallel
```

Why this order:

- Java gives language strength.
- Python gives scripting, backend API, async, data, and production service breadth.
- SQL gives backend data confidence.
- SpringBoot turns Java + SQL into production services.
- Microservices teaches distributed architecture and senior trade-offs.
- DSA keeps coding rounds warm.

## Folder Rules

- Keep every topic inside its main track folder.
- Put original/imported source files under `_Originals/`.
- Keep one index file per track.
- Add new platinum/gold sheets to the matching topic folder, not the top level.
